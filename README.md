# PulseAI - AI-Powered News Discovery Platform

PulseAI continuously scans the internet in real-time and delivers personalized news, stories, trends, and discussions that users care about most.

## 🎯 Core Vision

An intelligent news discovery platform that combines cutting-edge AI with real-time data aggregation to help users discover important stories before they become mainstream.

## 👥 Target Users

- Football fans
- Content creators
- Journalists
- Sports analysts
- General news consumers

## ✨ Key Features

### 1. **AI News Aggregation**
- Collect from websites, blogs, social media, YouTube, Reddit, podcasts
- Automatic duplicate detection
- Event-based story clustering

### 2. **Personalized Feed**
- Interest-based customization
- AI behavioral learning
- Unique per-user experience

### 3. **AI Story Summaries**
- Concise summaries with key facts
- Contextual relevance explanation

### 4. **Story Timeline**
- Track story evolution
- First report to latest confirmation

### 5. **Trending Intelligence**
- Rapid topic detection
- Trend scoring system
- Viral potential estimation

### 6. **Creator Mode**
- Instagram captions
- Reels scripts
- YouTube Shorts scripts
- X/Twitter threads

### 7. **Smart Search**
- Topic-based search
- AI-powered summaries (not just links)

### 8. **Breaking News Alerts**
- Real-time notifications
- Custom alert filtering

### 9. **Trust Score**
- Source reliability rating
- Confidence levels per story

### 10. **AI Recommendation Engine**
- Smart story suggestions
- Engagement-based learning

## 🏗️ Tech Stack

### Frontend
- **Next.js** - React framework with SSR/SSG
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Modern utility-first styling
- **Mobile-first** responsive design

### Backend
- **Supabase** - PostgreSQL + Auth + Realtime
- **PostgreSQL** - Primary database
- **Authentication** - Secure user management

### AI & ML
- Story clustering algorithms
- News summarization engine
- Recommendation system
- Trend detection
- Content generation
